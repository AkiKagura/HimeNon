# HimeNon
姫の音、midiファイルをテキストにする簡単なプログラムです。

## 使用方法
### 必要なモジュール
・mido  
・numpy

### 手動変更必要項目
・入力、出力パス  

## 出力内容フォーマット
・1小節に4拍というのがほとんどの楽譜のルールですが、本プログラムでは前の小節が後の小節の長さを占めている場合がありますので注意してください  
・数値は音高、`#`は8度上昇  
・数字の左の記号は音符の長さを表し、デフォルトは8分音符。  
 `:`で4分音符、`.`で8分音符、`'`で16分音符、`"`で32分音符の長さの表しでございます  
・数字の右側の記号は8度の変化を表し、8度ごとに矢印を追加します
